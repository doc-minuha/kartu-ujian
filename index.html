Panduan Penggunaan Aplikasi

Langkah 1: Isi Identitas Sekolah

Mulailah dengan mengisi semua informasi mengenai sekolah dan jenis ujian pada halaman 'Identitas Sekolah'. Data ini akan muncul di kop dan bagian tanda tangan kartu ujian.

Langkah 2: Input Data Siswa

Masukkan data semua siswa peserta ujian. Anda bisa menambahkannya satu per satu atau mengimpor data massal menggunakan template CSV yang disediakan untuk efisiensi.

Langkah 3: Atur Mata Pelajaran

Daftarkan semua mata pelajaran yang akan diujikan. Daftar ini akan menjadi pilihan saat Anda mengatur jadwal ujian di langkah berikutnya.

Langkah 4: Buat Jadwal Ujian

Tetapkan jadwal ujian untuk setiap kelas. Pilih tanggal, lalu tambahkan mata pelajaran beserta jam mulai dan selesainya. Jadwal ini akan otomatis tercetak di kartu setiap siswa sesuai kelasnya.

Langkah 5: Pratinjau dan Cetak

Langkah terakhir adalah melihat pratinjau kartu ujian. Anda dapat memfilter berdasarkan kelas dan kemudian mencetak semua kartu yang diperlukan. Hasil cetak dioptimalkan untuk kertas F4.

Backup & Restore Data

Backup Data

Simpan semua data aplikasi (Identitas Sekolah, Siswa, Mata Pelajaran, dan Jadwal) ke dalam satu file JSON. Simpan file ini di tempat yang aman sebagai cadangan.

Restore Data

Pulihkan data dari file backup (.json). Peringatan: Proses ini akan menimpa semua data yang ada saat ini di aplikasi.

Zona Berbahaya

Tindakan ini akan menghapus seluruh data yang tersimpan di browser Anda dan mengembalikan aplikasi ke kondisi awal. Tindakan ini tidak dapat diurungkan.

Untuk mengimpor data, ikuti langkah-langkah berikut:
